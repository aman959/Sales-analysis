## Project Description
An end-to-end Power BI capstone project that transforms raw retail
transactional data from the Sample Superstore dataset into an interactive,
professional sales analytics dashboard.
The dashboard enables business decision-makers to monitor KPIs,
analyze regional performance, identify profitable and loss-making
products, and study year-over-year sales trends.
## Features
- 4 KPI Cards: Total Sales, Total Profit, Total Orders, Profit Ratio
- Line Chart: Sales trend over time (monthly/yearly)
- Bar Chart: Regional sales comparison
- Pie Chart: Category-wise sales distribution
- Bar Chart: Profit by Sub-Category with conditional formatting
- Top 10 Products ranked bar chart
- Interactive slicers: Region, Category, Date Range
- DAX Measures: DIVIDE, DISTINCTCOUNT, SAMEPERIODLASTYEAR
## Tools Used
- Microsoft Power BI Desktop
- Power Query Editor
- DAX (Data Analysis Expressions)
- Sample Superstore CSV (Kaggle)
## Dataset
Source: Kaggle - Sample Superstore Dataset
Rows: ~9,994 | Columns: 21 | Period: 2014-2017
## How to Use
1. Download Power BI Desktop (free): powerbi.microsoft.com
2. Clone this repository or download the ZIP
3. Open the file: SalesAnalysisDashboard.pbix
4. Use the slicers to filter by Region, Category, or Date
5. Hover over charts for detailed tooltips
